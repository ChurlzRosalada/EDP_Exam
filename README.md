# EDPPREFINALEXAM

To get this project running on your local machine and contribute, follow these steps:

### 1. Clone the repository
```bash
git clone [https://github.com/leightonluy/EDPPREFINALEXAM.git](https://github.com/leightonluy/EDPPREFINALEXAM.git)
cd EDPPREFINALEXAM
```
##Dependencies need to install:
```
npm i
npm install react-router-dom

npm install @phosphor-icons/react
```
